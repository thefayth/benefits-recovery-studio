# Security Policy

## Supported Surface

This repository contains only the public project surface for Benefits Recovery Studio. It does not contain the private engine, private records, credentials, forms, filings, or operational infrastructure.

## Reporting

If you notice exposed private data, credential-like text, or a safety issue in the public materials, contact Faith through FaithCheltenham.com. Do not open public issues containing sensitive details.

## Non-Public Materials

The following are intentionally excluded from this repository:

- medical and benefits records;
- forms, notices, packets, and filing receipts;
- private prompts, agent instructions, and workflows;
- credentials, tokens, cookies, and account details;
- family information and identity records;
- operational infrastructure and deployment systems.
