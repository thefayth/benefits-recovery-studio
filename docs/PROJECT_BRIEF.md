# Project Brief

## Project

Benefits Recovery Studio

## Public Summary

Benefits Recovery Studio is a public-facing explanation of a private administrative recovery system. It shows how complex benefit, health coverage, and document-response workflows can be made calmer, auditable, and easier to review without exposing the underlying records.

## Core Promise

Public surface. Private engine. Receipts always.

## What This Repository Is

- a project overview;
- a workflow map;
- a privacy boundary;
- a visual identity package;
- a WordPress page draft for FaithCheltenham.com;
- an ownership and commercial-use boundary.

## What This Repository Is Not

- not a medical record archive;
- not a benefits filing packet;
- not a data room;
- not legal, medical, or benefits advice;
- not the full operational system;
- not an open source release.

## Audience

Potential collaborators, civic-tech partners, accessibility advocates, institutions, product evaluators, funders, journalists, and aligned builders.
